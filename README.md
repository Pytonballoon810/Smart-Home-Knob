# Smart-Home-Knob

## Secrets

These are the secrets that may need to be set in [secrets.h file](/firmware/secrets.h).
| Secret | value | required |
| --- | --- | --- |
| `WIFI_SSID` | WiFi SSID of the network | :negative_squared_cross_mark: |
| `WIFI_PASSWORD` | WiFi password of the network | :white_check_mark: |
| `MQTT_USER` | MQTT username for the broker | 
| `MQTT_PASSWORD` | MQTT password for the broker |
